# Mutual Fund Analytics Platform

## Project Overview
The Mutual Fund Analytics Platform is an end-to-end data analytics project developed as part of the Bluestock Fintech Capstone. The project analyzes mutual fund data using Python, SQLite, SQL, and Power BI to generate meaningful investment insights.

## Features
- Data Ingestion
- Data Cleaning
- SQLite Database
- Exploratory Data Analysis (EDA)
- Performance Analytics
- Advanced Analytics (Sharpe Ratio, VaR, CVaR)
- Power BI Dashboard
- Mutual Fund Recommendation System

## Tech Stack
- Python
- Pandas
- NumPy
- SQLite
- SQL
- Power BI
- Git & GitHub
- Jupyter Notebook
